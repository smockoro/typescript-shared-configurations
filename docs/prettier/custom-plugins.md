# カスタムフォーマットの方法/カスタムプラグイン

- カスタムフォーマットルールの作成
- デフォルト対応していない言語のフォーマット

後者は言語それぞれでデファクトのものを利用すればよく、prettierにこだわる理由は無いためここには記載しない。
